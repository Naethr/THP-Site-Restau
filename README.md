# Neo Bistrot Lab

Simple static front-end project for practicing DOM manipulation with vanilla JavaScript.

## Features

- Custom tabs: Home, Menu, Photos
- Random restaurant menu generator
- Exit-intent popup
- Draggable photo gallery
- Responsive restaurant-style layout

## Tech

- HTML
- CSS
- Vanilla JavaScript

No backend, no JavaScript framework, no external JS library.

## Run

Open `index.html` directly in a browser.
